# Textures Folder

## Purpose
Contains Monet-inspired texture PNGs (light/dark variants) used in backgrounds, cards, and overlays.

## File Overview

| File | Description |
| --- | --- |
| `brush_crosshatch_light.png` / `brush_crosshatch_dark.png` | Fine crosshatch overlay for cards. |
| `brush_soft_light.png` / `brush_soft_dark.png` | Soft wide brush strokes for depth. |
| `dapple_light.png` / `dapple_dark.png` | Subtle dappled lighting accents. |
| `edge_bleed_light.png` / `edge_bleed_dark.png` | Paper edge bleeding effect for page background. |
| `mist_wash_light.png` / `mist_wash_dark.png` | Broad watercolor wash base. |
| `paper_fiber_light.png` / `paper_fiber_dark.png` | Paper grain foundation texture. |
| `vignette_haze_light.png` / `vignette_haze_dark.png` | Corner vignette haze overlay. |
| `water_ripple_soft_light.png` / `water_ripple_soft_dark.png` | Gentle water ripple banding. |
