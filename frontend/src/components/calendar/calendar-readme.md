# calendar-readme

## Purpose
Calendar-specific UI components for the rolling week view and detail drawer.

## File Overview
| File / Module | Description |
| --- | --- |
| `CalendarWeekView.tsx` | Seven-day rolling columns with an all-day row, scrollable card lists, and drag interactions. |
| `CalendarDetailDrawer.tsx` | Slide-in drawer showing event or todo details with recurrence scope controls. |
