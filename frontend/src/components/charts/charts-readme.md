# Charts Folder

## Purpose
Contains Recharts-powered visualisations for HRV, resting heart rate, sleep, and training load trends.

## File Overview

| File | Description |
| --- | --- |
| `HRVChart.tsx` | Line chart for heart-rate variability trend. |
| `RHRChart.tsx` | Line chart for resting heart rate trend. |
| `SleepChart.tsx` | Bar chart for nightly sleep duration. |
| `LoadChart.tsx` | Area chart for rolling training load. |
