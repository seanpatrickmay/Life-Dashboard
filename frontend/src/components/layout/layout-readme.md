Layout wrappers and navigation shells that provide page-level structure and Monet-themed backgrounds.

## File Overview

| File | Description |
| --- | --- |
| `PageBackground.tsx` | Viewport background with Monet textures that now computes bridge top/width/left plus shoreline-island gutters to keep UI cards above the fold. |
| `PageShell.tsx` | Navigation shell and content surface used by all pages. |
