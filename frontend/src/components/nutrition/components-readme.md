# Nutrition Components Folder

## Purpose
Shared building blocks for the nutrition tracker page: Claude chat panel, food manager, goals panel, and dashboards.

## File Overview

| File | Description |
| --- | --- |
| `ClaudeChatPanel.tsx` | Inline chat UI to converse with Claude and show logged entries. |
| `FoodManager.tsx` | Manual food list/editor for nutrient definitions. |
| `GoalsPanel.tsx` | Shows nutrient goals and allows inline edits. |
| `NutritionDashboard.tsx` | Daily summary and 14-day history visualizations. |
