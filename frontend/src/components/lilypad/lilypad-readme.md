# Lilypad Components Folder

## Purpose
Holds the scene-integrated lily pad card components used on the Dashboard, Insights, and Nutrition pages for displaying metrics and interactive forms.

## File Overview

| File | Description |
| --- | --- |
| `LilyPadCard.tsx` | Base lily pad sprite + content wrapper that positions and centers card content on the pad. |
| `LilyPadsInsights.tsx` | Insight and trend cards for the insights scene. |
| `LilyPadsNutrition.tsx` | Nutrition chat, menu, and dial cards rendered on lily pads. |
