# Theme Folder

## Purpose
Monet-inspired theming primitives: palette definitions, global styles, ThemeProvider logic, and Recharts overrides.

## File Overview

| File | Description |
| --- | --- |
| `GlobalStyle.ts` | Defines CSS variables, global font settings, accessibility rules. |
| `monetTheme.ts` | Theme palette and light/dark theme objects. |
| `ThemeProvider.tsx` | Theme context managing light/dark/system modes. |
| `rechartsTheme.ts` | Helpers to derive chart colors from the current theme. |
| `styled.d.ts` | Extends styled-components typing with Monet theme types. |
