# Life Dashboard

Self-hosted Garmin + Vertex AI dashboard delivering Monet-inspired wellness insights.

## Project Highlights

- **FastAPI backend** for Garmin ingestion and Vertex AI readiness summaries.
- **React frontend** styled after Monet's Water Lilies with pixel typography.
- **PostgreSQL storage** for activities, daily metrics, and AI insights.
- **Dockerized stack** for simple local deployment.

See `docs/setup.md` for instructions and `docs/pipeline.md` for architecture details.
