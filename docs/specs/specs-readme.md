# Specs Folder

## Purpose
Centralized product/design specs for the Monet dashboard scene, covering copy, layout rules, sprite prompts, and motion cues.

## File Overview

| File | Description |
| --- | --- |
| `layout-rules-spec.md` | Defines geometry variables and safe-zone heuristics for the scene-aware UI. |
| `copy-spec.md` | Narrative copy deck for surface labels, tags, and UI voice. |
| `theme-motion-spec.md` | Details time-of-day palettes plus motion choreography (sun/moon, boat, particles). |
| `lilypad-scenes-spec.md` | Documents lily pad density modes and feature toggles for seasonal scenes. |
| `pixel-sprite-prompts.md` | Prompt references for generating pixel assets (bridge, pads, blossoms, etc.). |
